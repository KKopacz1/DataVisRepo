Summary: Latitude vs. X

The oerall purpose of this project was to visualize temperature changes as you approach the equator. To complete this task, we utilized the OpenWeatherMap API to create a dataset on selected cities. We then used latitude and longitude to randomly generate a list of locations.


Once the dataset was created, we then plotted weather vs. latitude. Additional factors we accounted for included cloudiness, wind speed, temperature, and humidity.
