Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on selected cities. We used latitude and longitude to randomly generate a list of 545 cities.


After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we accounted for included: cloudiness, wind speed, temperature, and humidity. This site provides the source data, as well as explanations and descriptions of any correlations witnessed.
